# Product Managers를 위한 무료 Claude Code 코스
원문: https://ccforpms.com

## 한 줄 요약
PM이 Claude Code를 실제 업무에 적용하도록, 툴 안에서 직접 수행하는 실습형 무료 과정이다.

## 핵심 내용
- 학습 방식은 영상 강의가 아니라 Claude Code 내부에서 과제를 수행하는 인터랙티브 진행이다.
- 시작 순서는 설치 → 저장소 클론 → `/start-1-1` 실행이며, 이후 단계별 안내를 따라 학습한다.
- 주요 역량은 파일 작업, parallel agents, custom sub-agents, `CLAUDE.md` 기반 project memory, `@-mentions`, 이미지 분석이다.
- PM 실무 적용 범위는 회의록/리서치 정리, PRD 작성과 다각도 리뷰, 데이터 분석, 경쟁 전략 수립까지 포함한다.
- 과정은 Module 0(준비), Module 1(기초), Module 2(고급 PM 실습)로 구성되며 문서형 레퍼런스를 함께 제공한다.

## 실습/실행 단계
1. Claude Pro 또는 Max를 준비하고 Claude Code를 설치한다.
2. 코스 자료를 다운로드하고 저장소를 클론한 뒤 `/start-1-1`로 시작한다.
3. Nimbalyst, Obsidian, VS Code 중 하나로 작업 환경을 구성해 실시간 파일 변화를 확인한다.
4. parallel agents로 다수 파일을 병렬 처리하고, 역할별 sub-agent(예: Engineer, Executive, User Researcher)를 만들어 리뷰한다.
5. 고급 모듈에서 PRD 작성, 퍼널/실험 분석, 경쟁사 조사와 전략 발표 자료 작성을 수행한다.

## 기억할 포인트
- 코딩이나 터미널 경험이 없어도 자연어 지시 중심으로 학습 가능하다.
- 권장 소요 시간은 약 10~12시간이고, 기본적인 PM 실무 이해가 필요하다.
- 제작자는 Carl Vellotti이며 Anthropic 공식/후원 과정은 아니다.
