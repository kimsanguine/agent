# 1.1: TaskFlow에 오신 것을 환영합니다
원문: https://ccforpms.com/fundamentals/welcome

## 한 줄 요약
이 모듈은 TaskFlow 세계관과 학습 방식을 익히고, `claude`와 `/start-1-1`로 첫 실습을 시작하게 해주는 오리엔테이션이다.

## 핵심 내용
- 과정의 중심은 읽기가 아니라 Claude Code에서 PM 업무를 직접 수행하는 실습이다.
- 학습은 두 축으로 구성된다: 인터랙티브 모듈(슬래시 커맨드로 시작) + 웹 Reference Guide(복습/참고용).
- TaskFlow는 원격 협업 팀을 위한 가상 프로젝트 관리 SaaS이며, 학습자는 Senior Product Manager 역할을 맡는다.
- 사용자 페르소나는 3가지다: Enterprise Admin(Sarah), IC Engineer(Mike), Team Lead(Alex).
- 주요 폴더 구조는 `company-context/`, `lesson-modules/`, `your-work/`, `CLAUDE.md`이며, `.claude/`는 숨김 폴더로 후속 모듈에서 다룬다.
- 모듈은 보통 15~30분 내외로 진행되며, 전체 과정은 개인 페이스로 학습 가능하다.

## 실습/실행 단계
1. 저장소가 없다면 먼저 Module 0.2를 따라 `claude-code-pm-course`를 clone한다.
2. 프로젝트 폴더에서 터미널을 열거나, 터미널에서 `cd ~/Documents/claude-code-pm-course`를 실행한다.
3. `claude`를 입력해 Claude Code를 연다.
4. `/start-1-1`을 실행해 본 모듈을 시작한다.
5. 진행 중 permission prompt가 뜨면 작업 흐름이 끊기지 않게 허용 설정으로 진행한다.
6. `Read(...)`, `Bash(...)` 같은 로그는 내부 동작이므로 의미를 몰라도 괜찮다.

## 기억할 포인트
- 모듈은 순서대로 진행하는 것이 가장 효과적이다.
- 코딩 경험이 없어도 학습 가능하게 설계되어 있다.
- Module 1 이후부터는 실제 PM 업무에 같은 패턴을 적용할 수 있다.
- 빠르게 넘기기보다 직접 수행하고 복습하는 방식이 중요하다.
