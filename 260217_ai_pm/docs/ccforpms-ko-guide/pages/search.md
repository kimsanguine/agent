# 검색
원문: https://ccforpms.com/search

## 한 줄 요약
이 페이지는 코스 전체를 빠르게 찾기 위한 Pagefind 기반 Search 사용법과, 검색 실패 시 대체 탐색 방법을 안내합니다.

## 핵심 내용
- Search 박스로 강의 전 페이지를 찾을 수 있고, 검색은 Pagefind의 client-side 방식으로 동작합니다.
- 더 정확한 탐색을 위해 phrase 검색, 복수 키워드 검색, 결과 섹션 확인을 권장합니다.
- 원하는 내용을 못 찾으면 키워드를 바꿔 재검색하고, Company Context와 Getting Started를 순서대로 확인하면 좋습니다.

## 실습/실행 단계
1. Search 입력창에 찾고 싶은 주제어를 입력합니다.
2. 결과가 많으면 phrase 검색이나 복수 키워드(예: PRD, template)로 범위를 좁힙니다.
3. 결과에 표시된 섹션 정보를 보고 관련 모듈 페이지로 이동합니다.
4. 여전히 없으면 동의어로 다시 검색하고, 사이드바 탐색과 Company Context, Getting Started를 확인합니다.

## 기억할 포인트
- Pagefind 검색은 브라우저에서 바로 처리되어 사용이 가볍습니다.
- 검색 정확도는 키워드 조합 방식에 크게 영향을 받습니다.
- 검색이 막히면 코스 내 구조 탐색을 병행하는 것이 가장 빠릅니다.
