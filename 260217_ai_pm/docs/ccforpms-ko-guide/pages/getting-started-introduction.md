# 0.0: PM을 위한 Claude Code 소개
원문: https://ccforpms.com/getting-started/introduction

## 한 줄 요약
Claude Code는 PM의 반복 실행 업무를 자동화해, 전략적 판단과 의사결정에 더 집중하게 해주는 실행형 AI 도구다.

## 핵심 내용
- Claude Code는 내 컴퓨터의 파일/폴더를 직접 다루며, 대화만으로 문서 작성·데이터 분석·정리 작업을 수행한다.
- 핵심 기능은 File operations(@), Visual analysis, Parallel agents, Custom sub-agents, Project memory(CLAUDE.md)로 구성된다.
- 필요 조건: Claude Pro 또는 Max, Mac/Windows/Linux, 설치 시간 약 15분.
- 학습 시간은 총 10~12시간 수준이며, 주당 5~10시간의 업무 절감 효과를 목표로 한다.
- 활용 영역은 PRD, roadmap, 고객 피드백 분석, ticket 변환, release notes, stakeholder 커뮤니케이션, 경쟁/시장 리서치까지 넓다.
- 일반 챗봇 대비 강점은 지속 메모리, 실제 실행(파일 생성/명령 수행), 다문서 동시 처리, parallel 작업, Git 및 MCPs 연동이다.
- 사례로 Monday.com PM이 Reddit 댓글 34,000건을 단시간에 분석해 제품 우선순위 설정에 반영했다.
- 핵심 사고 전환은 명령어 학습이 아니라 원하는 결과를 설계하고 오케스트레이션하는 방식이다.

## 실습/실행 단계
1. Claude Pro/Max와 작업 환경(Mac/Windows/Linux)을 준비한다.
2. Module 0.1에서 설치를 완료한다.
3. Module 0.2에서 Claude Code를 실행하고 코스 저장소를 clone한다.
4. 첫 대화를 통해 터미널 기반 협업 흐름을 익힌다.

## 기억할 포인트
- PM에게 중요한 것은 코딩 자체보다 업무 orchestration 역량이다.
- Claude Code의 본질은 조언 제공을 넘어 실행과 지속 컨텍스트를 함께 제공하는 점이다.
- CLAUDE.md와 custom sub-agents를 활용하면 반복 업무 생산성이 크게 올라간다.
