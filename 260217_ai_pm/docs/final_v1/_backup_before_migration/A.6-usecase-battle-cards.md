# 유즈케이스: 경쟁사 배틀 카드 라이브러리 구축

> **PM 역량**: 세일즈 인에이블먼트 / 경쟁 분석 / GTM 전략
> **난이도**: P → L
> **관련 문서**: [5.2-discovery-competitive-analysis.md](./5.2-discovery-competitive-analysis.md), [4.4-custom-skills.md](./4.4-custom-skills.md)

---

## 왜 PM에게 필요한가?

세일즈 팀에서 가장 많이 요청하는 것 중 하나가 "경쟁사 배틀 카드"입니다. "HeyGen이 이런 기능 있다는데 우리는?" "Synthesia랑 비교해서 어떻게 포지셔닝해야 해?"라는 질문이 매주 반복됩니다.

문제는 배틀 카드가 금방 구식이 된다는 것입니다. 경쟁사는 매달 새 기능을 출시하고, 가격 정책을 바꾸고, 새 고객 사례를 발표합니다. PM이 수동으로 최신 상태를 유지하기는 사실상 불가능합니다.

Claude Code는 **경쟁사 정보 수집 → 구조화 → 배틀 카드 생성 → 정기 업데이트**를 자동화합니다.

---

## 상황

당신은 AI Avatar 서비스의 PM입니다. 세일즈 VP가 "주요 경쟁사 3곳(HeyGen, Synthesia, D-ID)에 대한 배틀 카드를 만들어달라"고 요청했습니다. 다음 주부터 Enterprise 세일즈 캠페인이 시작됩니다.

---

## 기존 방식 vs Claude 방식

```
기존 (3~5일):
1. 경쟁사 웹사이트 수동 조사 (가격, 기능, 고객 사례)
2. G2/Capterra 리뷰 읽기
3. Reddit, Twitter 검색
4. 스프레드시트에 비교표 정리
5. Google Slides에 배틀 카드 형식으로 작성
6. 세일즈팀 피드백 반영
→ 작성 완료 후 2주면 이미 구식

Claude (반나절 + 자동 업데이트):
1. 경쟁사별 데이터 수집 → 배틀 카드 초안 생성 (2시간)
2. PM 리뷰 + 내부 인사이트 추가 (2시간)
3. 월간 자동 업데이트 설정
```

## HeyGen 배틀 카드

### 1. 제품 개요
핵심 기능:
- ✅ AI 아바타 생성 (100+ 프리셋) [heygen.com, 2025-02-20]
- ✅ 실시간 아바타 스트리밍 [2024년 12월 출시]
- ✅ 다국어 립싱크 (40+ 언어) [heygen.com/features]
- 🔵 우리에게 없는 것: Interactive Avatar API (실시간 대화형)
- 🔴 우리가 더 나은 것: 감정 표현 정밀도, 커스텀 아바타 품질

최근 업데이트 (6개월):
- 2024.12: Interactive Avatar 2.0 출시 [프레스 릴리즈]
- 2025.01: Enterprise SSO 지원 [체인지로그]
- 2025.02: 비디오 번역 API v3 [개발자 문서]

### 2. 가격 정책
| 플랜 | 가격 | 크레딧 | 제한 |
|------|------|--------|------|
| Free | $0 | 1분/월 | 워터마크 |
| Creator | $29/월 | 15분/월 | 720p |
| Business | $89/월 | 30분/월 | 1080p + API |
| Enterprise | Custom | 무제한 | SLA + 전용 지원 |
[출처: heygen.com/pricing, 확인: 2025-02-20]

⚠️ PM 확인 필요: Enterprise 가격은 공개되지 않음.
   세일즈팀에서 입수한 정보가 있으면 추가해주세요.

### 3. 고객 프로필
타겟: 중소기업 마케팅팀, 교육 기관, 미디어 에이전시
대표 고객: [공개 사례 기반]
- Amazon (교육 콘텐츠)
- Accenture (내부 교육)
- 특화 산업: EdTech, 마케팅

### 4. G2 리뷰 분석 [G2.com, 2025-02-20]
평점: 4.6/5 (856 리뷰)
긍정 테마: 사용 편의성, 빠른 생성 속도
부정 테마: 아바타 자연스러움 부족, 긴 영상 품질 저하

### Step 2: 배틀 카드 형식으로 변환

```
Terminal 입력:
> 수집한 데이터를 세일즈팀이 바로 쓸 수 있는 배틀 카드 형식으로 변환해줘.

  배틀 카드 구조:
  ## 한 눈에 보기
  - 30초 엘리베이터 피치: "HeyGen 대비 우리가 나은 이유"
  - 핵심 차별화 3가지

  ## When We Win (우리가 이기는 상황)
  - 어떤 고객 니즈에서 우리가 우세한지
  - 실제 Win 사례

  ## When We Lose (우리가 지는 상황)
  - 어떤 상황에서 경쟁사가 우세한지
  - 대응 전략 (어떻게 약점을 완화할지)

  ## Landmine Questions (세일즈 킬러 질문)
  - 경쟁사에게 물어보게 할 질문 5개
  - 예상 답변과 우리의 카운터

  ## Objection Handling (반론 대응)
  - "HeyGen이 더 싸잖아" → 대응
  - "HeyGen은 실시간 아바타가 되는데" → 대응
  - "HeyGen이 더 많은 언어를 지원하는데" → 대응

Claude Code 응답:

# 🏷️ 배틀 카드: vs HeyGen

## 한 눈에 보기

**30초 피치**: "HeyGen은 빠르고 저렴한 아바타 영상 생성에
강하지만, Enterprise급 품질과 감정 표현이 필요한 경우 우리가
압도적으로 우세합니다. 특히 브랜드 아이덴티티를 반영한 커스텀
아바타에서 HeyGen은 프리셋 의존도가 높습니다."

**핵심 차별화 3가지**:
1. 🎭 감정 표현 정밀도 — 6가지 기본 감정 + 미세 표정 조합
2. 🖌️ 커스텀 아바타 품질 — 10분 영상으로 브랜드 전용 아바타 생성
3. 🔒 Enterprise 보안 — On-premise 배포 옵션 (HeyGen은 SaaS only)

## When We Win 🏆

| 고객 니즈 | 우리 강점 | 근거 |
|----------|----------|------|
| 브랜드 일관성이 중요 | 커스텀 아바타 품질 우위 | A/B 테스트: 브랜드 인지도 +23% |
| 감정 전달이 핵심 | 미세 감정 표현 지원 | 교육 콘텐츠 완강률 +18% |
| 보안/컴플라이언스 | On-premise 배포 가능 | 금융/의료 Enterprise 필수 |

## When We Lose ⚠️

| 경쟁사 우세 상황 | 대응 전략 |
|----------------|----------|
| 빠른 PoC가 필요한 경우 | → 우리도 15분 Quick Start 데모 강조 |
| 실시간 대화형 아바타 | → 로드맵에 Q3 예정, 현재는 비동기 품질 강조 |
| 가격 민감 중소기업 | → ROI 계산기 제공, TCO 비교 |

⚠️ PM 추가 필요: 실제 Win/Lose 사례는 CRM 데이터에서
   보강해주세요. 위 내용은 공개 정보 기반 추정입니다.

## Landmine Questions 💣

경쟁사와 비교 평가 중인 고객에게 제안할 질문:

1. "아바타의 감정 표현을 얼마나 세밀하게 조절할 수 있나요?"
   → HeyGen: 제한적 (프리셋 기반) / 우리: 6축 감정 조절
2. "커스텀 아바타 생성에 어떤 데이터가 필요하고, 소유권은?"
   → HeyGen: 플랫폼 종속 / 우리: 고객 소유 + 이식 가능
3. "On-premise 배포가 가능한가요?"
   → HeyGen: 불가 / 우리: 가능
4. "1시간 이상 장편 영상의 품질 일관성은?"
   → HeyGen: 장편에서 품질 저하 보고 있음 (G2 리뷰)
5. "SLA에서 보장하는 업타임과 응답 시간은?"
   → HeyGen: 공개 SLA 없음 / 우리: 99.9% + 전용 지원

## Objection Handling 🛡️

**"HeyGen이 더 싸잖아요"**
→ "맞습니다. 단순 아바타 영상이 목적이라면 HeyGen이 비용
효율적입니다. 하지만 Enterprise에서 중요한 것은 단가가 아니라
TCO입니다. 수정 횟수, 브랜드 가이드라인 준수율, 내부 승인
프로세스까지 포함하면 우리의 '한 번에 완성' 품질이 실제로
더 경제적입니다."

**"HeyGen은 실시간 아바타가 되는데요?"**
→ "실시간 아바타는 인상적인 기능이지만, 현재 주요 사용
사례를 보면 대부분 사전 제작 콘텐츠입니다. 실시간이 핵심
요구사항이시라면 구체적인 유즈케이스를 들려주시면 우리
로드맵과 매칭해보겠습니다."
```

## PM 판단 포인트

### ❓ "Claude가 만든 배틀 카드를 그대로 세일즈에 줘도 되나?"

```
반드시 PM이 보강해야 하는 부분:

1. [내부 데이터] CRM에서 실제 Win/Lose 사례 추가
   → Claude는 공개 정보만 사용, 실제 딜 데이터는 PM만 접근 가능

2. [세일즈 피드백] 현장에서 실제로 듣는 반론
   → "이런 반론은 본 적 없어" vs "이 반론은 매일 듣는다"

3. [로드맵 정보] "우리가 지는 상황"의 대응에 로드맵 포함 여부
   → 외부에 공유 가능한 수준으로 조절

4. [가격 전략] 경쟁사 가격 대비 포지셔닝 결정
   → 할인 정책, 번들링은 세일즈 전략의 영역

5. [톤 조절] 경쟁사를 직접 비방하지 않도록
   → "HeyGen은 품질이 낮다" ❌
   → "우리는 이런 상황에서 더 나은 결과를 제공합니다" ✅
```

### ❓ "배틀 카드를 최신 상태로 유지하려면?"

```
자동 업데이트 전략:

1. 월 1회 정기 업데이트:
   - 경쟁사 웹사이트 가격/기능 변경 감지
   - 새로운 G2/Capterra 리뷰 수집
   - 프레스 릴리즈/블로그 업데이트 확인

2. 이벤트 트리거 업데이트:
   - 경쟁사 신기능 출시 시
   - 우리 기능 업데이트 후 차별화 포인트 변경 시
   - 세일즈팀에서 새로운 반론 보고 시

3. 검증 프로세스:
   - 자동 수집 → PM 리뷰(10분) → 세일즈 승인 → 배포
```

## 실습 과제

### Level 1 (J): 경쟁사 1곳 배틀 카드
자신의 제품과 가장 자주 비교되는 경쟁사 1곳에 대해 배틀 카드를 생성하세요. 자동 수집된 정보의 정확도를 수동으로 검증하세요.

### Level 2 (P): 3곳 + 세일즈 피드백 통합
경쟁사 3곳 배틀 카드를 생성하고, 세일즈팀 인터뷰(최소 2명)에서 얻은 실전 인사이트를 추가하세요. 월간 자동 업데이트를 설정하세요.

### Level 3 (L): 배틀 카드 시스템
배틀 카드 생성 → 업데이트 → 배포 → 피드백 수집의 전체 시스템을 설계하세요. Slack 봇으로 세일즈팀이 실시간으로 경쟁 정보를 요청할 수 있게 하세요.
