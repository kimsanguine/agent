# 8.3 Cowork Enterprise: 조직 확대를 위한 플랫폼 (2026년)

## 🔗 하네스 엔지니어링과의 연결

이 Part는 하네스 엔지니어링의 **실행 루프(Execution Loop)**를 엔터프라이즈 수준으로 확장하는 가이드입니다.

### 연결 지점

- **실행 루프 — 엔터프라이즈 확장**: SSO, 감사 로그, 팀 권한 관리로 Repository Harness를 조직 전체에 체계적으로 적용합니다.
- **헌법 거버넌스**: 프로젝트 기반 자동화와 권한 관리로 헌법(CLAUDE.md)의 일관된 적용을 보장합니다.

**참고**: [dependency-map.md](./dependency-map.md)의 실행 루프(Execution Loop) 매핑 섹션을 먼저 읽으면 이 Part의 내용이 더 명확해집니다.

## 개요

**Cowork Enterprise**는 2026년 2월부터 제공되는 **조직 수준의 AI 협업 플랫폼**입니다. SSO, 감사 로그, 팀 권한 관리, 프로젝트 기반 자동화로 기업의 AI 도입을 체계화합니다.

## Part A: Cowork Enterprise 핵심 기능

### 기본 기능 (Cowork vs Enterprise)

| 기능 | Cowork | Cowork Enterprise |
|------|--------|------------------|
| **IDE** | 개인/팀 용 | 조직 수준 |
| **사용자 수** | 1-50명 | 50명+ |
| **인증** | 이메일 | SSO (Google, Azure, Okta) |
| **감사 로그** | 기본 | 상세 (모든 작업 추적) |
| **권한 관리** | 기본 (팀) | 고급 (RBAC, 부서별) |
| **감시 & 규정** | 없음 | ✅ SOC2, HIPAA 준비 |
| **데이터 상주** | 다중 리전 | 단일 리전 선택 가능 |
| **API 접근** | 제한적 | 무제한 |
| **지원** | 이메일 | 전담 account manager |
| **가격** | $20/사용자/월 | $50/사용자/월 (최소 50명) |

### Cowork Enterprise의 5가지 핵심

```
┌─ 보안 & 규정
│  ├─ SSO (Single Sign-On)
│  ├─ 감사 로그 (모든 작업 기록)
│  ├─ 데이터 암호화 (저장/전송)
│  ├─ 2FA/MFA 강제
│  └─ GDPR/HIPAA 준수
│
├─ 조직 관리
│  ├─ 부서별 팀 구성
│  ├─ 역할 기반 접근제어 (RBAC)
│  ├─ 예산 관리 (부서별 할당)
│  ├─ 사용자 온보딩 자동화
│  └─ 오프보딩 (권한 회수)
│
├─ 워크플로우 자동화
│  ├─ Projects (마스터 프로젝트 보드)
│  ├─ Scheduled Tasks (조직 레벨)
│  ├─ Dispatch (원격 제어 승인)
│  ├─ Integration API (Slack, Jira)
│  └─ Webhooks (자동 트리거)
│
├─ 모니터링 & 분석
│  ├─ 대시보드 (팀별 생산성)
│  ├─ 실시간 메트릭 (작업 상태)
│  ├─ 비용 분석 (부서별 지출)
│  ├─ 성과 리포트 (월간)
│  └─ 이상 탐지 (비정상 활동)
│
└─ 규제 & 컴플라이언스
   ├─ 감사 준비 (Audit Trail)
   ├─ 정책 강제 (AI 사용 정책)
   ├─ 데이터 거버넌스 (내부/외부)
   ├─ 보존 정책 (로그 보관 기간)
   └─ 인증서 관리 (SOC2, ISO27001)
```

## Part B: Enterprise 도입 단계별 가이드

### Phase 1: 초기 평가 (2주)

```
1. 요구사항 정의
   ├─ 사용자 수: 50명? 200명? 1,000명?
   ├─ 부서: 몇 개 팀?
   ├─ 규제 요구: HIPAA? GDPR? SOC2?
   └─ 통합: Slack? Jira? Notion?

2. 파일럿 설정
   ├─ 파일럿 팀 (10-20명)
   ├─ 2주 트라이얼
   ├─ 기본 사용 케이스 3개 테스트
   └─ 피드백 수집

3. ROI 계산
   ├─ 기존 시간 낭비 측정
   ├─ Cowork로 절감 예상치
   ├─ 인력 비용 절감
   └─ 품질 향상 가치화
```

### Phase 2: 조직 구성 (1개월)

```
1. SSO 설정
   ├─ Azure AD, Google Workspace, Okta와 연동
   ├─ 사용자 정보 동기화
   ├─ 2FA/MFA 강제 설정
   └─ 테스트 & 검증

2. 팀/부서 구조 정의
   ├─ 조직도 기반 팀 생성
   ├─ 역할별 권한 할당 (Admin, Member, Viewer)
   ├─ 부서별 예산 설정
   └─ 승인 워크플로우 정의

3. 정책 수립
   ├─ AI 사용 가이드라인
   ├─ 데이터 보안 정책
   ├─ 감사 로그 보존 기간
   └─ 규제 준수 체크리스트
```

### Phase 3: 롤아웃 (2개월)

```
Wave 1: 파일럿 팀 확대 (30명)
├─ 완전 교육 제공
├─ 담당자 지정
├─ 주간 체크인
└─ 초기 문제 해결

Wave 2: 초기 도입 (100명)
├─ 자동 온보딩 시스템
├─ 팀 관리자 교육
├─ Slack 통합 (공지)
└─ 월간 업데이트

Wave 3: 전사 배포 (전체)
├─ 전사 공지 및 교육
├─ 마이그레이션 지원
├─ 레거시 도구 통합
└─ 완전 전환 완료
```

## Part C: 실제 기업 사례

### 사례 1: Series B SaaS 회사 (100명)

**상황:**
- 팀: 개발팀 30명, 마케팅 15명, 운영 10명, 기타 45명
- 문제:
  - 각 팀이 독립적으로 Claude 사용 (정책 없음)
  - 감사/규제 요구 (개인정보 침해 우려)
  - 중복된 작업 자동화 (각 팀이 별도로 개발)
  - 비용 통제 불가

**Cowork Enterprise 도입 효과:**

```
Before (수동 관리):
─────────────────
• 팀별 자동화 도구: 7개 (각자 개발)
• PM 시간 낭비: 팀별로 노하우 공유 안 됨
• 규제 리스크: 감사 로그 없음
• 비용: 팀별로 다름 (통제 불가)

After (Cowork Enterprise):
──────────────────────────
1단계: 조직 구조 설정 (2주)
  ├─ SSO: Azure AD 연동
  ├─ 팀 구조: 4개 부서별 팀
  └─ 역할: Admin 5명, Member 95명

2단계: 프로젝트 기반 자동화 (1개월)
  ├─ "마케팅 콘텐츠 생성" 프로젝트 (공유)
  ├─ "지원 티켓 분류" 프로젝트 (공유)
  ├─ "재무 리포트" 프로젝트 (공유)
  └─ 각 팀이 재사용 가능

3단계: 거버넌스 적용 (2개월)
  ├─ 승인 워크플로우 (PM 승인 후 실행)
  ├─ 비용 트래킹 (팀별 $1,000/월 제한)
  ├─ 감사 로그 (모든 작업 기록)
  └─ 규제 준비 (SOC2 감사 통과)

결과:
──────
비용 절감:
  • 중복 개발 제거: 월 50시간 × 12팀 = 600시간 = $48,000/연
  • 자동화 최적화: 월 40시간 × 100명 = 4,000시간 = $320,000/연
  • API 비용 최적화: 월 $5,000 → $2,000 = $36,000/연
  • 총 절감: $404,000/연

품질 향상:
  • 규제 준수: SOC2 감사 통과
  • 보안: SSO + 2FA로 데이터 보호
  • 일관성: 조직 표준 정책 적용
  • 가시성: 실시간 대시보드로 모니터링

Cowork Enterprise 비용:
  • 100명 × $50/월 × 12 = $60,000/연

ROI = ($404,000 + 값어치 있는 효과) / $60,000 = 574% (1년 내)
```

### 사례 2: 금융 서비스 회사 (500명 이상)

**상황:**
- 고도 규제 산업 (GDPR, CCPA, 금융감시)
- 엄격한 보안 요구
- 다국적 팀 (EU, US, APAC)
- 감사 로그 필수

**Cowork Enterprise 도입:**

```
특화된 설정:
──────────

1. 보안 강화
   ├─ 데이터 상주: EU 리전만 (GDPR)
   ├─ MFA 강제: 모든 사용자 필수
   ├─ IP 화이트리스트: 회사 IP만 접근
   └─ DLP (Data Loss Prevention): 민감 정보 차단

2. 규제 준수
   ├─ 감사 로그: 3년 보관
   ├─ 컴플라이언스 보고서: 분기별 자동 생성
   ├─ 규정 정책: "금융 데이터는 외부 전송 금지"
   └─ 위반 감지: 비정상 활동 자동 알림

3. 조직 확대
   ├─ 부서: 12개 부서 × 40명 = 480명
   ├─ 역할: Admin 10명 + Manager 50명 + 팀원 420명
   ├─ 비용 센터: 부서별 $5,000/월 할당
   └─ 승인: 중요 작업은 2-레벨 승인

결과:
──────
• 규제 준수: SOC2 Type II 인증 획득
• 감사 효율: 연간 감사 비용 $100,000 → $20,000 (80% 절감)
• 운영 자동화: 월 100시간 효율성 향상
• 리스크 감소: 규제 위반 0건 (이전 3건/연)
```

## Part D: Cowork Enterprise 비용 분석

### 가격 모델

```
기본 요금:
├─ 사용자당: $50/월
├─ 최소: 50명 = $2,500/월
├─ 500명: $25,000/월
└─ 1,000명+: 협상 가능

추가 기능:
├─ 전담 Account Manager: +$3,000/월
├─ 커스텀 SSO/SAML: +$2,000/월
├─ 고급 감사 로그 (1년): +$5,000/월
├─ SLA 보장 (99.99%): +$2,000/월
└─ 데이터 거주지 선택: +$1,000/월

예시 구성:
100명 회사:
  • 기본: 100 × $50 = $5,000/월
  • Account Manager: +$3,000/월
  • 고급 감사: +$5,000/월
  • 합계: $13,000/월 = $156,000/연
```

### ROI 계산

```
일반 SaaS 회사 (100명):

절감 항목:
────────
1. PM 자동화 도구 통합: $30,000/연
   (Zapier, IFTTT 제거)

2. 협업 효율성: $50,000/연
   • 중복된 스크립트 제거
   • 공동 프로젝트 보드 (Asana 대체)
   • 자동 통합 (Slack, Jira)

3. 규제 준수: $40,000/연
   • 감사 시간 단축
   • 컴플라이언스 자동 보고

4. 개발 비용 절감: $60,000/연
   • 팀별로 따로 개발하던 자동화 제거
   • 모듈화된 프로젝트 재사용

총 절감: $180,000/연

Cowork Enterprise 비용: $156,000/연

ROI = ($180,000 - $156,000) / $156,000 × 100 = 15% (1년)
     = 3.2년 후 누적 ROI 100% 달성

추가 효과 (값어치 산정):
───────────────────
• 보안 강화: $50,000/연 (리스크 감소)
• 팀 생산성 향상: $200,000/연 (시간당 30분 절감)

총 효과: $430,000/연
최종 ROI = ($430,000 - $156,000) / $156,000 = 176% ✅
```

## Part E: Cowork Enterprise vs Claude Code

### 언제 어떤 것을 쓸까?

```
조직 규모별 권장:

1-5명 (스타트업):
└─ Claude Code 100%
   • Cowork 불필요 (협업 부담 없음)
   • 비용 효율 (인당 $0)

5-20명 (성장기):
├─ Claude Code (메인): 고속 개발
└─ Cowork (선택): 협업 필요 시
   • Cowork는 선택사항
   • Projects로 팀 동기화

20-50명 (확대기):
├─ Claude Code (고급): 전략/분석
└─ Cowork (메인): 팀 협업
   • 중급 팀원: Claude Code
   • 비기술팀: Cowork UI

50-200명 (구축기):
├─ Claude Code: 소수 전문가만
└─ Cowork Enterprise: 중심
   • 대부분 Cowork
   • Claude Code는 보조

200명+ (엔터프라이즈):
└─ Cowork Enterprise 100%
   • SSO, 감사, 규제 필수
   • Claude Code는 고급 사용자만
```

### 기능 비교

| 기능 | Claude Code | Cowork | Cowork Ent |
|------|-------------|--------|-----------|
| 터미널 기반 자동화 | ✅ | ❌ | ❌ |
| GUI 협업 | ❌ | ✅ | ✅ |
| 프로젝트 관리 | 기본 | 중급 | 고급 |
| 팀 권한 | ❌ | 기본 | ✅ 고급 |
| SSO | ❌ | ❌ | ✅ |
| 감사 로그 | ❌ | 기본 | ✅ 상세 |
| 비용 | 포함 | $20/user | $50/user |
