## 1. 프로젝트 개요

### 1.1 프로젝트 목표

LinkedIn 활동을 체계적으로 관리하고 성장시키기 위한 통합 대시보드 및 자동화 시스템

**핵심 가치 제안:**
- 📊 **성과 가시화**: 포스트/팔로워/참여율을 한눈에 파악
- 🤖 **AI 자동화**: 댓글 생성 및 타겟 관리 자동화
- 📈 **전략적 성장**: 데이터 기반 인사이트로 효율적 성장

### 1.2 타겟 사용자

- **1차 타겟**: 개인 브랜딩을 위해 LinkedIn을 활발히 사용하는 전문가 (개발자, PM, 디자이너 등)
- **2차 타겟**: 소규모 비즈니스 오너, 프리랜서, B2B 마케터
- **사용 빈도**: 주 3-5회, 1회당 10-20분

### 1.3 주요 Pain Points 해결

| Pain Point | 해결 방법 |
|------------|----------|
| LinkedIn 분석 데이터가 산발적 | 통합 대시보드로 모든 메트릭 한눈에 |
| 댓글 관리 비효율적 | AI 댓글 생성 + 승인 큐 시스템 |
| 타겟 네트워킹 전략 부재 | 타겟 프로필 관리 + 우선순위 시스템 |
| 콘텐츠 성과 비교 어려움 | 포스트별 성과 테이블 + 정렬/필터 |

## 2. 현재 구현 기능 (v1.1)

> 상태 기준: 2026-02-17 (P0 UI/UX 반영, 추천 타겟 패널/API 연동 완료, 실데이터 연동 준비)

### 2.1 대시보드 (3개 탭)

#### **Tab 1: Overview (개요)**
- **KPI 메트릭 카드 (4개)**
  - 총 팔로워 (증감 표시)
  - 평균 노출 (주간 비교)
  - 평균 참여율 (변화율)
  - 총 포스트 (주간 발행 수)
- **차트 섹션 (2열)**
  - 인게이지먼트 트렌드 차트 (Recharts 영역 차트)
  - 팔로워 성장 추이 (Recharts 라인 차트)
- **4주 성장 추세 텍스트**: AI 생성 인사이트

#### **Tab 2: Posts (포스트)**
- **포스트별 성과 테이블**
  - 컬럼: 포스트 내용, 노출, 참여율, 반응, 댓글, 날짜
  - 정렬 기능 (모든 컬럼 클릭 가능)
  - 필터 기능 (타입, 기간)
  - 포스트 타입 텍스트 배지 (예: `build_log`, `milestone`)
  - 참여율 색상 구분 (8%↑ 초록, 5-8% 파랑, 5%↓ 회색)

#### **Tab 3: Engagement (참여)**
- **댓글 큐 패널 (좌측)**
  - 상태별 필터 (대기/승인/게시/전체)
  - 통계 표시 (대기 5, 승인 3, 게시 12)
  - AI 댓글 아이템:
    - 타겟 이름, 톤(전문적/친근/질문형)
    - 품질 점수 (0-100)
    - 원문 미리보기
    - 액션 버튼 (승인/수정/거부)
  - 인라인 텍스트 수정 기능
- **타겟 프로필 패널 (우측)**
  - "+ 새 타겟 추가" 버튼
  - 타겟 프로필 카드:
    - 이름, 직함, 회사
    - 토픽 태그 (#AI, #ML)
    - 우선순위 (High/Medium/Low)
    - 참여 횟수, 최근 참여 날짜
    - 삭제 버튼
- **추천 타겟 패널 (우측)**
  - TRS 기반 추천 리스트 (Connect/Follow/Watch/Skip)
  - 프로필 DNA/PI 배지 표시
  - 액션: Connect, Follow, Add Target, Hide, Snooze, Skip

## 3. 사용자 니즈 분석

### 3.1 콘텐츠 성과 가시성 부족 ⭐⭐⭐⭐⭐

**Pain Point:**
- LinkedIn 네이티브 분석이 산발적이고 통합되지 않음
- "어떤 포스트가 잘됐는지" 비교하기 어려움
- 시간대별/주제별 패턴 파악 불가능

**현재 해결 수준:** ✅ **80% 해결**
- 포스트별 성과 테이블 (정렬 가능)
- KPI 메트릭 카드 (한눈에 보는 통계)
- 트렌드 차트

**향후 개선:**
- 📊 **AI 인사이트**: "이번 주 'AI 기술' 태그 포스트가 평균 대비 2.3배 높은 참여율"
- 🔍 **포스트 타입 분석**: 텍스트 vs 이미지 vs 비디오 성과 비교
- ⏰ **최적 게시 시간 추천**: "화요일 오전 9시에 게시하면 노출률 +34%"

---

### 3.2 댓글 관리의 비효율성 ⭐⭐⭐⭐⭐

**Pain Point:**
- 타겟 인플루언서 포스트에 일일이 찾아가서 댓글 달아야 함
- 어떤 댓글이 효과적인지 모름
- 댓글 품질 일관성 유지 어려움

**현재 해결 수준:** ✅ **70% 해결**
- 댓글 큐 시스템 (AI 생성 → 승인 → 게시)
- 품질 점수 표시
- 타겟 프로필 관리

**향후 개선:**
- 🤖 **자동 댓글 스케줄링**: 승인된 댓글을 최적 시간에 자동 게시
- 📈 **댓글 성과 추적**: "이 타겟에 단 댓글로 프로필 방문 +12명"
- 💡 **댓글 템플릿 라이브러리**: 주제별 효과적인 댓글 패턴 저장

---

### 3.3 네트워킹 전략 부재 ⭐⭐⭐⭐

**Pain Point:**
- "누구와 연결해야 하는지" 모름
- 기존 네트워크 활용도가 낮음
- 팔로워 성장이 느리고 전략적이지 않음

**현재 해결 수준:** ✅ **65% 해결**
- 타겟 프로필 관리
- 참여 횟수 추적
- 우선순위 설정
- TRS 기반 추천 타겟 패널(Connect/Follow/Watch/Skip)
- 추천 액션 반영(Connect/Follow/Add Target/Hide/Snooze/Skip)

**향후 개선:**
- 🎯 **AI 타겟 발굴**: "당신의 포스트에 자주 반응하는 2촌 연결 추천"
- 🔗 **네트워크 맵**: 영향력 있는 연결의 시각화
- 📊 **타겟별 ROI**: "이 타겟과의 상호작용으로 팔로워 +5명 획득"

---

### 3.4 콘텐츠 아이디어 고갈 ⭐⭐⭐⭐

**Pain Point:**
- 매일 무엇을 올릴지 막막함
- 트렌드 파악 어려움
- 나만의 스타일/주제 찾기 어려움

**현재 해결 수준:** ⚠️ **20% 해결**
- 성장 추세 텍스트에서 힌트만 제공

**향후 개선:**
- 💡 **AI 콘텐츠 제안**: 과거 성과 기반으로 "이번 주 추천 주제 3가지"
- 🔥 **트렌딩 토픽**: 당신의 업계에서 급상승 중인 키워드
- 📝 **콘텐츠 캘린더**: 일주일 포스팅 계획 + 리마인더

---

### 3.5 일관성 유지의 어려움 ⭐⭐⭐⭐

**Pain Point:**
- 바쁘면 LinkedIn 활동 중단 → 알고리즘 불이익
- 정기적 포스팅 습관 만들기 어려움
- 댓글/참여도 꾸준히 하기 힘듦

**현재 해결 수준:** ❌ **0% 해결**

**향후 개선:**
- 📅 **자동 포스팅 스케줄러**: 한 번에 일주일치 작성 → 자동 게시
- ⏰ **일일 알림**: "오늘의 타겟 3명에게 댓글 달 시간입니다"
- 🎖️ **성과 스트릭**: "7일 연속 활동 중! 계속 유지하세요"

---

### 3.6 측정 불가능한 ROI ⭐⭐⭐⭐

**Pain Point:**
- "LinkedIn 활동이 실제 비즈니스에 도움이 되는가?" 불확실
- 리드 전환, 매출 연결 어려움
- 시간 대비 효과 측정 불가

**현재 해결 수준:** ⚠️ **30% 해결**
- 참여 메트릭만 제공

**향후 개선:**
- 💰 **리드 추적**: "이 포스트로 3명이 웹사이트 방문 → 1건 상담 요청"
- 📊 **시간 투자 분석**: "주 2시간 활동으로 월 평균 5건 리드 생성"
- 🎯 **목표 설정 & 추적**: "이번 달 목표: 팔로워 +100명 (현재 진행률 65%)"

### Phase 3: Advanced Features (2-3개월) 🎯

**목표**: 차별화된 고급 기능으로 플랫폼 경쟁력 강화

| 기능 | 우선순위 | 난이도 | 예상 시간 |
|------|---------|--------|----------|
| **자동 포스팅 스케줄러** | P2 | High | 2주 |
| **리드 추적 시스템** | P2 | High | 3주 |
| **네트워크 맵 시각화** | P2 | High | 2주 |
| **콘텐츠 캘린더** | P2 | Medium | 1주 |
| **경쟁자 벤치마킹** | P2 | Medium | 1주 |

#### 3.1 자동 포스팅 스케줄러
- 일주일치 콘텐츠 한 번에 작성
- 최적 시간에 자동 게시 (Buffer/Hootsuite 스타일)
- 이미지 업로드 지원

#### 3.2 리드 추적 시스템
